# What's in there?

SpiderPhy is a study carried on arachnophobic subjects. They were exposed to a variaty of visual stimuli, ranging from neutral objects to pictures of -- pretty scary -- spiders. Several measurments were made:

- ECG: measure of cardiac electric activity;
- Gaze: measure of where the subjects were looking during the experiment;
- ...
